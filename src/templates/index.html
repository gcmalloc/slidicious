Slidicious

Slidicious is an automated slide building service. It build your slide from markdown format to html and pdf. It then upload the result to this site.

How to setup slidicious

To make slidicious work, simply point the github hook url to {{url_for('github_hook')}}. This will trigger the build everytime you push the repository to gitub.

Your compiled slides can be found in /static/{username}/repositoryname.html for the html slide and /static/{username}/repositoryname.pdf for the pdf slides.
